# Data_Science_pjs

## Описание
В репозитории представлены проекты курса ["Data Science Specialist"](https://practicum.yandex.ru/data-scientist/) (Яндекс.Практикум).

## Основные инструменты и навыки, полученные при обучении:
- Языки: Python, SQL
- Анализ данных: библиотеки Pandas, NumPy, SciPy, Statsmodels
- Визуализация: Matplotlib, Plotly, Seaborn
- Построение дашбордов: Tableau
- А/В-тестирование
- Работа с гипотезами: постановка, проверка
- Машинное обучение: библиотеки Scikit-learn, XGBoost, CatBoost, LightGBM
- Кластеризация данных: DBSCAN, k-means, Hierarchical
